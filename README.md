RecruitSmart AI: Intelligent Hiring Ecosystem
RecruitSmart AI is a Streamlit-based recruitment automation platform designed to bridge the gap between high-volume applications and quality hiring. 
By leveraging advanced text extraction and skill-mapping algorithms, the system allows HR teams to process multiple candidate resumes against specific Job Descriptions (JD) in seconds.

Features:
Batch Processing: Upload 4–5 (or more) PDF resumes simultaneously for rapid evaluation.

AI-Powered Skill Detection: Automatically extracts and categorizes Technical and Soft skills from unstructured PDF data.

Dynamic Match Scoring: Calculates a percentage-based compatibility score by comparing candidate profiles directly against JD requirements.

Skill Gap Analysis: Provides a visual breakdown of "Skills Found" vs. "Missing Skills" to help recruiters identify training needs or non-negotiable gaps.

Interactive Leaderboard: A data-driven ranking page featuring interactive Plotly charts and tabular summaries for final candidate selection.
